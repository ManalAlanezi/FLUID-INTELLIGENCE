# FLUID-INTELLIGENCE
A real-time Fluid Dynamics Simulator that bridges the gap between Software Engineering and Industrial Physics.
What is this?
This isn't just a dashboard; it's a small physics engine. I built this to show how we can use code to predict fluid behavior in industrial pipelines. By calculating things like Reynolds Number and Pressure Drop in real-time, it helps identify potential risks like turbulence or erosion before they happen.

Why I built it:
I wanted to solve a real problem in the energy sector: making complex pipe data easy to understand and actionable for engineers.

Tech used:

Physics: Navier-Stokes & Darcy-Weisbach equations.

Frontend: HTML5, Tailwind CSS, and Chart.js for data visualization.

Logic: JavaScript for real-time computational analysis.
